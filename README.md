# 🔐 Penetration Testing on DVWA

## 📌 Project Overview

This repository contains a complete **Penetration Testing** assessment performed on **DVWA (Damn Vulnerable Web Application)**. The objective of this project was to simulate real-world web application attacks, identify security vulnerabilities, exploit them in a controlled laboratory environment, and provide detailed remediation recommendations.

The assessment follows a structured **Vulnerability Assessment and Penetration Testing (VAPT)** methodology aligned with industry standards and **OWASP Top 10** security guidelines.

---

# 🎯 Objectives

* Perform web application penetration testing in a controlled lab environment
* Identify vulnerabilities aligned with the OWASP Top 10
* Exploit security flaws to demonstrate real-world impact
* Document findings with risk ratings and remediation recommendations
* Strengthen practical understanding of ethical hacking methodologies

---

# 🔍 Vulnerabilities Identified

The following vulnerabilities were identified and tested:

* SQL Injection (SQLi)
* Cross-Site Scripting (XSS – Reflected & Stored)
* Command Injection
* Broken Authentication
* Cross-Site Request Forgery (CSRF)
* File Upload Vulnerabilities
* Brute Force Attacks
* Security Misconfiguration

Each vulnerability assessment includes:

* 📖 Vulnerability Description
* ⚙️ Exploitation Steps
* 📸 Proof of Concept (PoC)
* 📊 CVSS Risk Rating
* 🛡️ Remediation Recommendations
